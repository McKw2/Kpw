# Kpw

Marketing website scaffold for Kpw.

## Run locally

1. Start a static server:
   - `python3 -m http.server 4173`
2. Open `http://localhost:4173` in your browser.

## Project files

- `index.html` - one-page site layout and content
- `styles.css` - responsive styling and theme
- `script.js` - mobile nav toggle and contact form UX
